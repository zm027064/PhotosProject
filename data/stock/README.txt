This directory is for stock photos that will be loaded automatically into the "stock" user's "stock" album.

IMPORTANT REQUIREMENTS:
- You MUST put between 5-10 photos in this directory (no fewer, no more)
- Photos will stay in this workspace directory for graders to test
- Supported formats: .jpg, .jpeg, .png, .gif, .bmp
- Use medium-sized photos to keep the project size reasonable

The photos will be:
1. Loaded when the application starts
2. Stored under username "stock" (password="stock" or no password)
3. Available in an album named "stock"
4. Visible as examples to all users of the application

To use:
1. Copy 5-10 photos into this directory
2. Launch the application
3. Log in as user "stock" to verify photos loaded correctly
4. Other users can reference these stock photos

Note: These photos stay in the workspace, while photos added by regular users can be loaded from anywhere on their computer.
